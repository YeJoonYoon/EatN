Name: EatN(N개만 먹어) project
(Developed by Python 3.7)

주의사항 
- 녹화하는 창에 커서를 움직여 +표시가 되도록 하고, 
   그 상태로 q를 누르면 종료할 수 있습니다. 
- exe파일이 안 열릴 경우 main.py 파일을 직접 열어주세요.
- python파일을 열 경우, 필요한 library는 mediapipe, opencv가 되겠습니다. 
- exe파일을 끄고 새로 실행할 때는 video dir에 있는 영상을 모두 지워주세요.

기본적인 실행 방법
1. 실행을 누른다. (손님이 들어옴)
2. 녹화창이 켜지고 손모양을 인식해서 가져간 개수를 센다. (손님이 물건을 고름)
3. 녹화창이 꺼진다. (손님이 계산대에 도착함)
4. 손님이 주장하는 개수를 입력한다. (손님이 계산을 시도함)
5. 개수에 따라 경고 메시지가 주어지거나 그냥 보내줌. 
6. video directory에 영상이 저장됨.
